INSTRUCTION:

LANCEMENT:
	Pour lancer le jeu double cliquez sur le fichier "colletonpoint.exe"

EN JEU:
	1| Cliquez sur 'Start' pour lancer le chronomètre
	2| Cliquez sur l'image pour poser les gommettes
	3| Quand vous avez recouvert toutes les têtes cliquez sur 'Stop'
	4| Recommencez si vous avez envie d'améliorer votre score en appuyant sur 'Reset'
	5| Pour fermer la fenêtre appuyer sur 'Quitter'
	
PARTAGEZ VOTRE SCORE:
	Pour partagez votre score faites une capture d'écran (touche 'Impr écran' sur PC) et postez la sur
	Twitter avec #ColleTonPoint
